## Top 5 Things to Know Before Planning Anything Else

1. **Golden Week is April 29–May 5.** Hotels triple in price. Trains sell out. Book immediately.
2. **Get a Suica card the moment you land.** It works for trains, buses, convenience stores, vending machines everywhere.
3. **Japan still runs on cash.** Many restaurants, ryokan, and shrines are cash-only. Use 7-Eleven ATMs.
4. **Do not tip.** Ever. It can be considered rude.
5. **You will walk 15,000–25,000 steps/day.** Comfortable shoes are not optional.
