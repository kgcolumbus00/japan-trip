## Top 5 Things to Know Before Planning Anything Else

1. **Buy the USJ Express Pass now** — usj.co.jp. April 29 is Golden Week, park will be packed. No Express Pass = waiting in 2-hour lines all day.
2. **Get a Suica card the moment you land.** Works for trains, buses, convenience stores, vending machines everywhere. Load ¥5,000 at the airport JR machine.
3. **Japan still runs on cash.** Many restaurants, shrines, and smaller shops are cash-only. Use 7-Eleven ATMs.
4. **Do not tip.** Ever. It can be considered rude.
5. **You will walk 15,000–25,000 steps/day.** Comfortable shoes are not optional.
